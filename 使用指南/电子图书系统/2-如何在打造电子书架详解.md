
## 如何在Obsidian设置书架为卡片样式
首先我们要知道，电子书架是通过dataview这个插件来实现的，但是dataview的原始的呈现形式只有list、table、calendar、task这4中形式，而电子书架需要将封面、作者、文件链接呈现在一张卡片上，例如这样

### 方式1：使用minimal主题
想要制作这样的卡片布局，需要用到minimal主题中的cards样式，如果你安装了minimal主题，那你可以在frontmatter中设置`cssclasses:cards`就可以了，当然如果你还需要更精细的卡片调整，需要设置更多的class，支持的样式可以参见[minimal主题文档](https://minimal.guide/features/helper-classes#Cards)


例如以下
```
cssclasses:
  - cards
  - cards-cover, cards-1-1, table-max, cards-cols-4
```

cards语句：代表dataview table样式调整为cards样式，这是必须的，否则其他样式不生效。
list-cards语句:将所有项目符号列表设置为卡片布局
cards-align-bottom语句：将卡片的最后一个元素与底部对齐
cards-cover语句:调整图像大小以填充定义的空间
cards-1-1语句:调整卡片中的图片比例为1:1
table-max语句：代表全宽表格
cards-cols语句：代表设置卡片列数

### 方式2：使用其他主题+css代码片段
1. 第一步：同以上方式1
2. 第二步：在`.obsidian>snippets`路径下，其中有一个minimalcards.css文件，我们需要将这个文件启动，启动方式为`设置-外观-css代码片段`

## 如何进一步在Obsidian主页DIY自己的电子书架
模板库中已经提前预设了一个书架，代码如下：
```
Table without id embed(link(imageData)) ,"![](" + imageData + ")" as Cover,
file.link as title, "*作者：" + author + "*" , "**进度：" + progress + "%**","评价：" + 评价 + "",description
From "Books"
Where contains(status, "在读") 
```
需要先读懂这些代码，才好进一步DIY
`table`：代表dataview展现形式为表格
`without id`：代表不需要文档id
`("![|50](" + imageData + ")")`：代表图片调用了frontmatter中的图片链接，并且设置图片大小为50px
`as`：代表将表格抬头重命名为xx
`file.link`：代表文档链接
`author`：代表作者（引用了frontmatter）
`From "Books"`代表查找的数据来源于`Book`路径
`Where` ：是排除语句，模板库设定为查询内容需要包含`status`为`在读`
知道了以上内容后，你就可以开始DIY自己的电子书架了
