
## ABCDE模块化人生管理系统快捷键
以下是win系统的设置，如果你是mac系统，则 Ctrl=cmd，Alt=Option；

| 快捷键          | 功能               | 备注                                                                  |
| ------------ | ---------------- | ------------------------------------------------------------------- |
| Alt+1        | 嵌入Meta-bind模板    | 用于嵌入Assistants中的模块                                                  |
| Alt+2        | 插入2分栏            | 适用于所有情况，支持第三方插件                                                     |
| Alt+3        | 插入3分栏            | 适用于所有情况，支持第三方插件                                                     |
| Alt+4        | 插入callout注释      | 自动化了输入流程，与鼠标右键Inset>callout效果一样                                     |
| Alt+A        | 插入Admonition     | 比软件自带的callout更强大                                                    |
| Alt+C        | 重组笔记             | 将当前笔记合并到其他笔记中                                                       |
| Alt+F        | 插入column分栏       | 适用于图片、文档分栏，或不支持第三方插件                                                |
| Alt+M        | 完形填空样式           | 选中文字，使用此命令，将选中文字替换为完形填空样式，例如<sub>这样</sub>                           |
| Ctrl+shift+A | 一键切换完形填空         | 一键切换（显示、隐藏）所有完形填空内容                                                 |
| Alt+P        | 插入批注             | 像word一样，在文档左侧、右侧（栏宽以外）插入批注                                          |
| Alt+Q        | 在当前文档插入模板        | 笔记抬头模块中“选择插入模板”按钮功能一致                                               |
| Alt+S        | 创建闪念笔记           | 自动化笔记流程                                                             |
| Alt+T        | 添加视频时间戳笔记        | 需要同时打开视频、笔记                                                         |
| Alt+W        | 批量嵌入库外文件         | 将指定路径的文件夹、文件批量嵌入当前文档光标位置                                            |
| Alt+X        | 查看今日需要详述的笔记      | 查看今日需要详述的笔记                                                         |
| Alt+Y        | 将带有标签的笔记移动至指定文件夹 | 此命令通过quickadd宏命令实现，移动文档排除了Assistants文件夹，但是需要*谨慎*移动（如果标签管理的好，此命令很有用） |
| Alt+shift+1  | 更新模块             | 请先更新使用指南，查看0-更新日志文档，再决定是否更新模块                                       |
| Alt+shift+3  | 更新使用指南           | 建议定期更新                                                              |
| Ctrl+F1      | 创建Inbox笔记        | 创建闪念笔记，与“主页”中的蓝色按钮功能一致                                              |
| Ctrl+F2      | 创建Project笔记      | 创建项目笔记，与“主页”中的蓝色按钮功能一致                                              |
| Ctrl+F3      | 创建Resource笔记     | 创建资源笔记，与“主页”中的蓝色按钮功能一致                                              |
| Ctrl+F4      | 创建Archive笔记      | 创建归档笔记，与“主页”中的蓝色按钮功能一致                                              |
| Ctrl+D       | 创建/打开今日日记        | 与左侧工具栏中的按钮功能一致                                                      |

以上快捷键涉及本系统的核心功能，因此建议记住这些快捷键：
- 使用以上快捷键，将提高本系统的使用效率和使用体验；
- 当然，为了提高使用体验，在[[homepage-主页|主页]]、[[homepage-侧页|侧页]]中，均设置了可视化功能按钮，按钮与快捷键具有同样的效果；
- 由于使用了不同的键盘，或许会导致快捷键不起作用，请排除是否外接键盘导致快捷键不起作用；
- 预设的快捷键或许与你电脑中自行安装的 app 快捷键冲突（目前尚未发现与系统快捷键冲突），若有冲突，请排除这些因素；

## 已知的快捷键冲突

- Alt(OPT)+2：在mac 系统中，与 pixpin 中的快捷键存在冲突，需要修改或关闭 pixpin 中的快捷键设置。
