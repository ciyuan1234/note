# 如何使用分栏（重要）
==❗❗❗本系统采用模块化设计，合理嵌入不同模块可以组装不同的功能、效果的模板，组装过程离不开分栏，因此分栏是使用模块的核心，一定要学会==
## 方式1：使用[[10-ABCDE模块化人生管理系统快捷键|ATL+2、ATL+3]]插入分栏布局
以下是代码示例：
```````
``````ad-flex 
title:习惯追踪图表 
collapse:close
color: 24,22,22
icon: apple
`````col
````col-md
flexGrow=1
===
这里插入分栏1
````

````col-md
flexGrow=1
===
这里插入分栏2
````
``````
```````
解释：这是admonition+columns插件联用实现的分栏方式（预制仓库使用了[[使用指南/00--通用说明（重要！！！）/6-如何解决点击卡片或者代码块会进入编辑模式问题|定制的admonition插件]]，使得预览模式下点击分栏内容不会进入编辑模式，因此==请勿更新admonition插件==，更新会导致整个仓库无法使用）
- ad-flex：使用了admonition的注释语法，这里可以是ad-note、ad-todo等，你可以在`设置>第三方插件>Admonition设置`中查看支持的类型
- title：输入标题（注：支持markdown语法）
- color: 这里支持16进制的颜色选择，分别代表red，green，blue，数字可以填0-255对三色进行调整
- icon: 支持修改图标，图标支持的类型，可以在`设置>第三插件>Admonition设置>新增>Admonition图标`中查找
- \`\`\`\`\`col 代表下面的内容（\`\`\`\`col-md包裹部分 ）以竖向分栏呈现，
- flexGrow=1：数字代表分栏的比例，数字越小，分栏的宽度越小<sup>💬</sup><span class="sidenote">（最细版本的cloumns插件支持更多样式设置，例如边框等）</span> 
- 折叠：使用折叠功能必须要有title

## 方式2：使用ATL+F快捷键插入自定义分栏
以下是代码示例：
`````
````col
```col-md
flexGrow=1
===
# Column 0
```
```col-md
flexGrow=1
===
# Column 1
```
````
`````

**⚠️注意**：此方式插入的分栏内容，或不兼容第三方插件，在阅读模式可能无法正常显示，例如meta bind按钮、输入框、tracker等，但是预览模式可正常显示。适用于文本、图片或者文本+图片的分栏。
