### 如何下载安装Obsidian 

#### PC端下载安装
​Obsidian 是一款私密且灵活的写作应用程序，可以适应您的思维方式。我们可以通过以下几个方式来下载安装软件： 
- 方式1：官方网站（[🌐www.obsidian.md](https://www.obsidian.md)）下载
	- 网站托管在国外，可能会有网络问题。
	- 请注意管网仅支持最新版本（公布版本）下载，意味着无法从该网站获取早先版本。
- 方式2：github（[🌐github.com](https://github.com/obsidianmd/obsidian-releases/releases))下载
	- github是一个版本控制系统的在线代码
	- 托管平台，意味着你可以在这个网站中获取到不同版本的Obsidian，访问该网站或许需要梯子[^1]。
	- 最新的版本并不意味着最好，相反，新版本或许存在更多的BUG，这将导致兼容性问题，尤其大量使用第三方插件的情况下，或导致插件不再兼容新版本，从而导致软件无法正常运行。
- 方式3：[🌐国内中文论坛](https://forum-zh.obsidian.md/t/topic/13718)置顶帖下载
	- 同样可以获取到不同版本的Obsidian安装包。
	- 相对于github下载方式，优点是无需魔法、梯子，缺点是提供的版本不全面，且不一定及时更新。

#### 移动端下载安装

##### 安卓系统
安卓、鸿蒙系统，可以通过应用市场或者网页论坛下载安装包
- 方式1：从应用市场下载安装（部分品牌手机未上架）
- 方式2：[🌐国内中文论坛](https://forum-zh.obsidian.md/t/topic/13718)下载apk安装包到手机，在手机中打开apk安装程序。
##### 苹果系统
由于安全策略等因素不一样，苹果系统目前只能通过App store下载并安装。
- 从App Store下载安装

### 下载安装第三方插件
下载安装第三方插件有多种方式、途径，
#### 1. **官方社区插件市场（推荐）**
- **适用场景**：官方推荐方式，插件自动更新，安全可靠。  
- **安装步骤**：
  1. 打开 Obsidian → 设置 → `社区插件` → `浏览`。
  2. 搜索插件名称 → 点击安装。
- **注意事项**：
  - 需要科学上网
 
#### 2. **通过 GitHub 手动安装**
- **适用场景**：插件未上架官方市场，或需要安装金丝雀版本。  
- **安装步骤**：
  1. 在 GitHub 搜索插件仓库（如 `obsidian-插件名`）。
  2. 下载 `Releases` 中的 `.zip` 或 `.tgz` 文件。或者分别下载`main.js`、`manifest.json` 和 `styles.css`三个文档。
  3. 解压后放入 Obsidian 插件目录：  
     - Windows: `VaultFolder/.obsidian/plugins/插件名/`  
     - macOS: `右键仓库 → 显示包内容 → .obsidian/plugins/插件名/`  
  4. 重启 Obsidian 并在设置中启用插件。
- **注意事项**：
- 需要网络能访问 GitHub（或需要科学上网）
  - 非社区插件插件无法通过第三方插件设置中的“检查更新”按钮进行更新。

#### 3. **国内镜像站（如 PKMer）**
- **适用场景**：无法访问官方市场时的替代方案。  
- **推荐平台**：
  - [PKMer 插件市场](https://pkmer.cn/market/)  
  - 其他可信社区（如少数派、知乎专栏等）。  
- **特点**：
  - 同步官方插件的最新版本（但可能存在延迟），无需科学上网。
- **风险提示**：
  - 仅从可信来源下载，避免恶意修改的插件。

#### 4. **其他途径**
- **网盘/分享群**：
  - 常见于插件作者或爱好者分享的压缩包。
  - 需注意文件安全性（扫描病毒后再安装）。
- **开发者模式**：
  - 适用于自行开发或调试插件，通过 `npm` 或 `git clone` 直接加载。
