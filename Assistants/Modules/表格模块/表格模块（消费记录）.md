```js-engine
const mb = engine.getPlugin('obsidian-meta-bind-plugin').api;

const tableOptions = {
	bindTarget: mb.createBindTarget('frontmatter', context.file.path, ['contactX']),
	tableHead: ['消费', '金额', '备注'],
	columns: [
		'INPUT[text(class(text-100)):scope^消费]',
		'INPUT[number:scope^金额]',
		'INPUT[text(class(text-100)):scope^备注]',
	],
};

const mountable = mb.createTableMountable(context.file.path, tableOptions);

mb.wrapInMDRC(mountable, container, component);

```

可以在其他markdown文件中输入/粘贴以下代码块以使用本模块；

[[表格模块（消费记录）]]
```
````
输入快捷操作：alt+1
粘贴快捷操作：右上角图标复制，然后粘贴
本模块被嵌套到其他文档中后；
- 点击`Add row`添加表格行数
- 添加行后，可以选择文件链接、选择日期、输入文本操作。添加的内容会绑定到前页frontmatter（文档属性/元数据/metadata）中，如果未显示，可以在`设置-编辑器-文档属性`开启它；
- 本模块通过js-engine插件调用meta bind接口实现的表格输入，因此需要修改的内容，可以点击右上角的</>符号，在columns所在行，使用meta bind输入语句实现表格调整；
