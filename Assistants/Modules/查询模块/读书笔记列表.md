# 📖读书笔记列表
###### 🟢在读
```dataview
list
From "Books"
WHERE contains(status, "在读") 
sort title asc
```
###### 🔴未读
```dataview
list
From "Books"
WHERE contains(status, "未读") 
sort title asc
```
###### 🟡已读
```dataview
list
From "Books"
WHERE contains(status, "已读")

```
