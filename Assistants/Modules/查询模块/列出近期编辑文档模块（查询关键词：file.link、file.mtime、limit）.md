```dataview
LIST WITHOUT ID file.link
sort file.mtime desc
limit 5
```

%%
📃使用说明：
可以在其他markdown文件中输入/粘贴以下代码块嵌入以使用本模板；
````右上角复制

```
````
输入快捷操作：alt+1

本模板被嵌套到其他文档中后；
- 根据需求修改`限制数量`
