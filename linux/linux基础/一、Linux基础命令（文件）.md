# 1.1Linux结构目录

---
## 1.1.1 Linux路径

**Windows：使用\表示层级关系
Linux：使用/表示层级关系（/在Linux系统中☞根目录，也☞当前目录）
绝对路径：** 以根目录为起点描绘路径
**相对路径：** 以当前目录为起点描绘路径
**路径表达符：** **.** 表示当前目录 
		    **..** 表示切换为上一级
		     **~** 表示HOME目录 
![](assets/一、Linux基础命令（文件）/file-20260202164710430.png)

**eg：** 在根目录下有一个文件夹test，文件夹内有hello.txt
   **回答：** ==/test/hello.txt==

# 1.2 Linux命令入门
--- 
## 1.2.1命令基础格式
```linux
命令 [-选项] [参数]
```
## 1.2.2 ls命令

```Linux
ls [-a -l -h] [Linux路径]
```
**功能：** ==列出当前目录下的内容== （终端默认工作区为用户home目录）
**参数：** -a  all的意思，列出全部文件（包含隐藏文件/文件夹）
       -l    以竖向列表形式展示，并且展示更多信息
       -h  与-l选项搭配使用，显示更合理的文件大小单位
==（选项可以结合使用 例如：ls -la）==
## 1.2.3 cd命令
```linux
cd  [无选项] [Linux路径]
```
**功能：** ==切换工作目录，若没有参数切换到HOME目录==
- cd命令无需选项，只有参数，表示切换目录
- cd命令不写参数，表示回到用户HOME目录
## 1.2.4 pwd命令

```Linux
 pwd [无选项] [无参数]
```
**功能:** ==查看当前工作目录路径==
## 1.2.5 mkdir命令
```Linux
mkdir [-p] [Linux路径（必填）]
```
**功能：** ==创建目录==
**选项：** -p 可用于一次性创建多层级目录，会自动创建不存在的父级目录

## 1.2.6 touch命令
```Linux
touch [无选项] [Linux路径]
```
**功能：** ==创建文件==
- ==Linux文件开头标识为d的是文件夹（终端中标蓝显示），-开头的为文件==
## 1.2.7 cat命令

```Linux
cat [无选项] [Linux路径]
```
**功能：** ==查看文件内容（一次性展示所有内容）==
## 1.2.8 more命令

```Linux
more [无选项] [Linux路径]
```
**功能:** ==查看文件内容（支持翻页）==
## 1.2.9 cp命令
```Linux
cp [-r] [参数1] [参数2]
```
**功能：** ==复制文件==
**选项：** -r 用于复制文件夹使用，表示递归
- **参数1** Linux路径，表示被复制文件或文件夹路径
- 参数2 Linux路径，表示复制文件路径
## 1.2.10 mv命令
```Linux
mv [参数1] [参数2]
```
 **功能：**==移动文件夹，也可用于文件改名（参数二不存在）==
 - **参数1** Linux路径，表示被移动文件或文件夹路径
- **参数2**  Linux路径，表示移动文件路径
## 1.2.11 rm命令
```Linux
rm [-r -f] [参数1 参数2 参数3...参数n]
```
**功能：** ==删除文件/文件夹==
- **选项：** -r 用于删除文件夹
		-f 强制删除
- **参数：** 表示要删除文件的路径，用空格隔开
