## 启动
```
vi/vim [文件名]（表示使用vim打开文件，如果没有会自行创建）
```
## 保存/退出
```
 ：wq（保存并退出）
 ：q（退出）
```
## 三种常用模式
![](../../Pasted%20image%2020260219183119.png)
**==默认：命令模式==**
1. h，j，k，l控制光标移动，左下上右
2. yy复制内容，p粘贴内容，dd删除内容，前面加数字可以表示执行次数，eg：3p=粘贴三次，2yy=复制两行内容
3. ^跳转到行首，$跳转到行尾
4. ：set number（显示行号）：set nonumber（关闭行号显示）
5. Ctrl+f（向前翻页）Ctrl+b（向后翻页）Ctrl+u（向上翻半页）Ctrl+d向下翻半页
6. G（跳转到文件最后一行）gg（跳转到文件最后一行）：【行号】（跳转到某一行）/【查找内容】（从光标所在位置向下查找）？【查找内容】（光标所在位置，向上查找）n（按顺序查找下一个目标）N（按相反顺序查找下一个目标） ==需注意查找时是区分大小写的，如果想要不区分大小写 **/【查找内容】\c  ==
