```
vi/vim
```

## 保存/退出
```
 ：wq（保存并退出）
 ：q（退出）
```
## 三种常用模式
![](../../Pasted%20image%2020260219183119.png)
==默认：命令模式==
1. h，j，k，l控制光标移动，左下上右
2. yy复制内容，p粘贴内容，dd删除内容，前面加数字可以表示执行次数，eg：3p=粘贴三次
==插入模式：i/I,a/A,o/O进入插入模式==（其中i，a表示在光标前后插入，I，A表示在行前行后插入）
![](../../Pasted%20image%2020260219183911.png)
==尾行模式：冒号“：”是尾行模式的前缀==
