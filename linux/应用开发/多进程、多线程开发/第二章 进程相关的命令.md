# 一、进程的相关命令
## 1、ps命令
1. **功能：** 显示当前进程的状态
2. **语法：** ps [options]
3. **常用选项：**
    -A：列出所有进程
    -e：与A功能类似
    -w：显示加宽，可以显示较多资讯
    -au：显示交详细的信息
    -aux：显示所有包含其他使用者的进程
  eg：ps -aux       ps -ef（和效果ps -aux一样）![](../../../assets/Pasted%20image%2020260404160845.png)
  ==**补充：**== ps -ef | grep “可执行文件名” （可以用于查找指定进程）
## 2、top命令
1. **功能：** 实时显示进程信息
2. **语法：** top 【选项】
3. **常用选项：** 
	-d: 改变显示的更新速度，或是在交谈式指令列（interactive command）按s
	-q：没有任何延迟的显示速度，如果使用这是superuser的权限，则top将会以最高优先级执行
	-c： 切换显示模式，共有两种模式，一是只显示中心档的名称，另一种是显示完整的路径与名称
	-S： 累积模式，将直接已完成或消失的子进程（dead child process）的CPU time累积起来
	-s： 安全模式，将交谈指令取消，避免潜在危机
	-i： 不显示任何限制（idle）或无用（zombie）的进程
	-n： 更新的次数，完成后将退出top
	-b：批次档模式，搭配“n”参数一起使用，可以用将top的结果输出到档案内![](../../../assets/Pasted%20image%2020260404174756.png)
