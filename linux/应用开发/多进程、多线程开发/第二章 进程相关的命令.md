# 一、进程的相关命令
## 1、ps命令
1. **功能：** 显示当前进程的状态
2. **语法：** ps [options]
3. **常用选项：**
    -A：列出所有进程
    -e：与A功能类似
    -w：显示加宽，可以显示较多资讯
    -au：显示交详细的信息
    -aux：显示所有包含其他使用者的进程
  eg：ps -aux       ps -ef（和效果ps -aux一样）![](../../../assets/Pasted%20image%2020260404160845.png)
  ==**补充：**== ps -ef | grep “可执行文件名” （可以用于查找指定进程）
