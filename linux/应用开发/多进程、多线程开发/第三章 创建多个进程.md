# 一、进程的创建
## 1.关于并发和并行
为了提高计算机执行任务的效率，一般采用的解决方案就是能让多个任务同时进行，这里可以使用**并发**和**并行**两种方式
- **并行：** 在cpu多核的支持下，实现物理上的同时执行
- **并行：** 在有限cpu核心的情况下（如果只有一个cpu核心），利用快速交换（时间片轮转）执行来达到宏观上的同时执行
【**==注意==**】并行是基于硬件完成的，而并发则通过软件算法来完成的，可以创建多个进程并发执行![](../../../assets/Pasted%20image%2020260404185920.png)
## 2.进程的创建
1. fork（）函数
	- 创建进程的函数需要调用fork（）函数，则会产生一个新的进程
	- 调用fork（）函数的进程叫做父进程，产生的新进程则为子进程
	- fork函数详解
		- **功能：** 创建一个子进程
		- **函数返回值：**
			- 成功：返回给父进程是子进程的pid，返回给子进程的是0
			- 失败：返回-1，并设置erno
```cpp
//头文件
#include<sys/types.h>
#include<stdio.h>
#include<unistd.h>

int main(void){
	pid_t cpid;
	cpid = fork();
	if(cpid==-11){
	perror("fork():";)
	}
}
```
