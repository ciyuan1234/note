# 一、文件IO

## 1.关于Linux文件io
- 在Linux系统下，用于对文件操作的库函数叫做文件I/O
- 主要包括open（）/close（）/read（）/write（）/lseek（）相应的的系统调用（准确来说是对系统调用的封装的库函数）
## 2.文件描述符
- 文件描述符，也可以叫文件标记符，用来标记文件的底层设备
- 文件描述符是一个**非负整数** ，当打开一个已存在的文件或者创建一个新文件时，内核向进程返回一个文件描述符
- 每个程序运行后，操作系统都会默认打开三个文件，**标准输入、标注输出、标准错误输出** ，文件描述符分别为**0，1，2** 
- 标准输入对应的设备一般为**键盘** 
- 标准输入与标准错误输出设备一般为**显示器** 
```cpp
#include<stdio.h>
#include<sys/types.h>
#include<sys/stat.h>
#include<fcntl.h>
#include<errno.h>
#include<unistd.h>

int main(int argc,char*argv[]){
write(1,"helloworld\n",11); //通过标准输出项屏幕打印helloworld
return 0;
}
```
